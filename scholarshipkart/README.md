# ScholarshipKart

Static ScholarshipKart landing page built with HTML5, CSS3, JavaScript, Bootstrap 5, and Bootstrap Icons.

## Structure

```text
scholarshipkart/
|-- index.html
|-- css/
|   `-- style.css
|-- js/
|   `-- script.js
|-- assets/
|   |-- images/
|   |-- icons/
|   `-- partners/
`-- README.md
```

## Notes

- Repeated cards, sliders, partner logos, services, and testimonials are generated from JavaScript arrays.
- Destination and testimonial sliders use vanilla JavaScript and CSS transforms.
- Partner rows use CSS keyframe transforms with pause on hover.
- No Swiper, GSAP, or external slider library is used.
